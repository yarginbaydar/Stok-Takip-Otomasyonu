# 🛍️ ButikStok - Web Tabanlı Stok ve Müşteri Yönetim Otomasyonu

Küçük ve orta ölçekli butikler, mağazalar ve işletmeler için geliştirilmiş, tam kapsamlı bir stok, satış ve müşteri takip sistemidir. Kullanıcı dostu arayüzü sayesinde işletmeler günlük operasyonlarını kolayca yönetebilirler.

## 🚀 Canlı Demo
Projeyi canlı olarak incelemek için tıklayın: **[http://yargin-stok.runasp.net/]**
*(Giriş Bilgileri -> Kullanıcı Adı: admin | Şifre: 1234)*

## 💻 Kullanılan Teknolojiler
* **Backend:** C#, ASP.NET Core MVC 
* **Veritabanı:** SQLite & Entity Framework Core (Code-First)
* **Frontend:** HTML5, CSS3, Bootstrap
* **Mimari:** MVC (Model-View-Controller)

## ✨ Öne Çıkan Özellikler
* 📦 **Ürün ve Kategori Yönetimi:** Barkod destekli ürün ekleme, stok takibi ve kategori bazlı listeleme.
* 👥 **Müşteri ve Cari Takip:** Müşteri kayıtları oluşturma, bakiye/borç durumu takip etme ve müşteriye özel notlar tutma.
* 🛒 **Satış İşlemleri (Transaction):** Ürün satışı yapma, stoktan otomatik düşme ve işlem geçmişini listeleme.
**Excel Entegrasyonu:** Tek tıkla Excel'den toplu ürün yükleme altyapısı.
* 🔒 **Güvenlik:** Cookie-based (Çerez tabanlı) kimlik doğrulama ile güvenli admin paneli.
